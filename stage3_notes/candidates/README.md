# Candidate Specs

Each JSON file in this directory represents one Stage 3 candidate bundle.

Required sections:

- `slug`
- `title`
- `family`
- `source_type`
- `task_run_ids`
- `literature`
- `evidence`
- `common_overrides`
- `task_overrides`

The helper CLI `python stage3_research.py new-candidate ...` creates a valid template.
